## Array - list of items

| type | time complexity |
|--| --|
| Lookup | O(1) |
| push | O(1) |
| insert | O(n) |
| delete | O(n) |

e.g.
// for a 32 bit system
// 4 * 4 = 16 bytes of storage
// ["Juice", "Apple", "Cheese", "Kale"]
| index | value |
| -- | -- |
| 0 | Juice |
| 1 | Apple |
| 2 | Cheese |
| 3 | Kale |
